# D&D Ultimate Tool - Lehetseges uj funkciok

## 1. Karakter es kampany
- Karakter build varazslo: lepeseinkent vezeti a jatekost faj, osztaly, felszereles es varazslatok kivalasztasan.
- Kampany naplo: session osszefoglalo, fontos NPC-k, helyszinek es emlekeztetok egy helyen.
- Level-up asszisztens: automatikusan ajanl valaszthato opciokat szintlepeskor.
- Karakter import/export: JSON alapu mentes es megoszthatosag.

## 2. DM es VTT eszkozok
- Initiative tracker: korok, statuszok, HP es mentodobasok kezelese.
- Encounter builder nehezseg becslessel: parti meret es szint alapjan.
- Dinamikus terkepek: token mozgas, fog-of-war, ping es rajzolas.
- Loot generator: ritkasag, szint es tematikus tabla alapjan.

## 3. Wiki es tartalom
- Osszetett kereses szurokkel: challenge rating, tipus, iskola, felszereles kategoria.
- Kedvencek es gyujtemenyek: sajat listak varazslatokbol, szornyekbol, itemekbol.
- Osszehasonlito nezet: ket osztaly, ket varazslat vagy ket item gyors osszevetese.
- Offline cache javitas: gyakran latogatott oldalak elore toltesse.

## 4. Kozoessegi funkciok
- Partiszoba meghivas linkkel: gyors becsatlakozas jatekosoknak.
- Jatekos profil statisztikak: karakterek szama, session activity, kedvenc osztaly.
- Session feedback: jatekosok ertekelhetik az alkalmat, DM jegyzetet kap.
- Shared notes: valos ideju kozos jegyzetfuzet parti szinten.

## 5. UX es technikai fejlesztes
- Onboarding flow fejlesztese: szemelyre szabott tutorial uj felhasznaloknak.
- Global command palette: gyors navigacio oldalak, karakterek es eszkozok kozott.
- Ertesitesi kozpont: in-app jelzesek meghivokrol, valtozasokrol, uj tartalmakrol.
- Hibakezeles es fallback allapotok egységesitese minden API oldalon.
